# 装备解析协议

## 解析与复制

装备解析工具右键装备可生成数据化装备 CD。解析成功是否消耗原装备由配置决定；解析失败消耗工具但不生成 CD。已数据化装备可在允许拷贝配置下消耗解析工具复制。

当前黑名单至少包含：`batnosehat`、`wagpunkhat`、`armorwagpunk`。带容器的装备不继承容器能力；虚拟装备会阻止自动卸甲。

## 头部与身体装备

头部、身体解析 CD 插槽后生成对应隐藏虚拟装备。虚拟装备不占原版头、身栏，清除角色限制并尽量继承原装备组件、标签和特殊效果。

- 防御与原版装备栏共同按最高吸收值结算。
- 容器、地面可交互和视觉覆盖不继承。
- 虚拟装备不消耗自身耐久，也不受生活协议耐久修复影响。

## 手部装备

手部解析 CD 不创建虚拟武器；多张协议可叠加。

- 继承攻击力、位面伤害和移速。
- 攻击距离大于等于 1 时，继承攻击力为原攻击力除以攻击距离；小于 1 时完整继承。
- 继承砍伐、开采、挖掘、锤击、捕捉等工具效率与强力开采。
- Kei 未持手部装备时，可用上述工具能力执行原版动作；不继承钓竿属性。

## 外观与兼容

解析 CD 可由配置选择显示原装备贴图/地面动画，或统一使用 `kei_analysis_cd`。原装备外观模式会尝试读取皮肤与动画；失败时回退统一 CD 外观。
