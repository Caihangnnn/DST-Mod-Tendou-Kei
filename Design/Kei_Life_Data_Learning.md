# Kei 生活数据学习设计

状态：当前方案已记录  
目标：定义 Kei 学习路线中“生活数据”的第一版获取方式。生活数据主要来自 NPC 经验、交易和好感度，用于提升 Kei 的生活质量能力。

## 一句话定位

生活数据是 Kei 从恒常世界的 NPC 社会经验中学习到的非战斗知识。它不直接提升战斗数值，而是改善种植、制作、维护等长期生存体验。

与战斗数据的区别：

```text
战斗数据：通过记录 Boss / 巨兽战斗样本获取，提升战斗能力上限。
生活数据：通过 NPC 教材、交易和好感度获取，提升生活质量能力。
```

第一版生活数据从三类 NPC 获取：

| NPC | 代码名 | 学习主题 | 教材 | 解锁能力 |
| --- | --- | --- | --- | --- |
| 月亮码头女王 / 猴子女王 | `monkeyqueen` | 初级植物学 | 《植物学的奥秘》 | 《初级园艺知识的应用》 |
| 流浪商人 | `wanderingtrader` | 交易技巧 | 《流浪商人的教诲》 | 《初级交易技巧》 |
| 寄居蟹隐士 / 珍珠 | `hermitcrab` | 修补维护 | 《珍珠的缝补技巧》 | 《初级缝补技巧》 |

## 总体流程

生活数据不采用战斗数据记录器的实验场流程，而采用 NPC 教材流程：

```text
与 NPC 交互 -> 获得知识教材 -> 使用教材解锁生活协议 -> 后续通过配方或状态触发能力
```

设计原则：

- 每个 NPC 对应一种生活知识主题。
- 教材是能力解锁材料，不是能力本身。
- 首次解锁需要消耗对应教材。
- 解锁后，相关能力改为消耗数据稳定性、电量或其他运行成本触发。
- 后续想到新的可学习知识时，可以继续按同一结构追加。

## 猴子女王：初级植物学

### NPC 定位

猴子女王是固定刷新在月亮码头的 NPC。原版交互中，她接受香蕉，并优先为玩家清除猴子诅咒；若没有诅咒，则给予蓝图等奖励。

Kei 从猴子女王处学习初级植物学知识，主题是高产香蕉林和植物生长经验。

### 教材获取

玩家制作《高产香蕉林种植经验》，并给予猴子女王。

暂定配方：

```text
《高产香蕉林种植经验》
材料：40 个香蕉 + 10 张莎草纸
```

猴子女王接受后给予：

```text
《植物学的奥秘》
```

### 能力解锁

玩家通过《植物学的奥秘》解锁：

```text
《初级园艺知识的应用》
```

首次解锁：

- 消耗 1 个《植物学的奥秘》。
- 只解锁能力，不产生实际效果。

首次解锁后：

- 《初级园艺知识的应用》配方改为消耗数据稳定性。
- 制作 / 触发后产生实际效果。

### 能力效果

触发后，以玩家为中心一定范围内，拥有多个生长阶段的作物或植物跳过一个生长阶段，直接进入下一个阶段。

建议边界：

- 只影响有正常成长阶段的作物或植物。
- 不影响特殊剧情植物、装饰物、Boss 相关实体或不可再生关键实体。
- 第一版只推进一个阶段，不直接成熟到最终阶段。

## 流浪商人：初级交易技巧

### NPC 定位

流浪商人是不断移动的 NPC。玩家靠近后可以进行交易，表现类似制作站。

原版交易特点：

- 有常驻交易。
- 有随机交易。
- 各项交易次数受库存限制。
- 每隔两天刷新一次。
- 月雹时会有特殊交易进货。

Kei 从流浪商人处学习交易知识，主题是材料估值、议价和高效制作。

### 教材获取

《流浪商人的教诲》通过流浪商人常驻交易获得。

暂定交易配方：

```text
《流浪商人的教诲》
材料：40 个金子 + 10 张纸
交易类型：流浪商人常驻交易
```

### 能力解锁

玩家使用《流浪商人的教诲》制作 / 解锁：

```text
《初级交易技巧》
```

### 能力效果

《初级交易技巧》表现为制作消耗优化：

- 每次制作后，都有概率使下一次制作获得“消耗减半”效果。
- 当获得消耗减半效果时，播放原版阅读《万物百科》时触发的音效和动画反馈。
- 下一次制作享受减半效果后，该效果消失。

建议将该效果命名为：

```text
交易灵感：下一次制作消耗减半。
交易回响：每次制作后，有概率获得交易灵感。
```

实现建议：

- 后续实现时，不一定直接修改配方消耗。
- 更稳妥的做法是正常制作后返还约一半可返还材料。
- 这样能减少对全局配方和其他模组配方的侵入。

## 寄居蟹隐士：初级缝补技巧

### NPC 定位

寄居蟹隐士居住在海上隐士小岛，拥有好感度系统。她本身类似制作站，玩家靠近后可解锁部分物品，并随着好感度提高解锁更多内容。

Kei 从寄居蟹隐士处学习修补和维护知识，主题是长期维护、耐久恢复和燃料补全。

### 教材获取

当寄居蟹隐士好感度达到 6 时，解锁《珍珠的缝补技巧》的交易配方。

暂定交易配方：

```text
《珍珠的缝补技巧》
材料：10 个空瓶子
解锁条件：寄居蟹隐士好感度达到 6
```

### 能力解锁

玩家使用《珍珠的缝补技巧》制作 / 解锁：

```text
《初级缝补技巧》
```

### 能力效果

《初级缝补技巧》用于恢复物品栏中拥有耐久值或燃料值的物品。

初始设想：

- 物品栏中有燃料值或耐久值的物品会随着时间逐渐恢复。
- 可突破原本耐久上限，最高达到 500%。
- 需要黑名单排除不适合恢复的物品。

当前建议采用触发式效果，而不是永久常驻效果：

```text
制作 / 触发《初级缝补技巧》
获得持续一天的“维护协议”
持续期间，物品栏中符合条件的物品逐渐恢复耐久或燃料
最高可恢复到 500%
```

采用触发式的原因：

- 降低长期后台扫描物品栏造成的性能压力。
- 避免常驻恢复与其他模组物品产生过多兼容问题。
- 500% 上限较强，适合作为持续一天的维护协议，而不是永久被动。

### 黑名单规则

第一版必须为可恢复物品增加黑名单。

建议排除：

- 不可替代物品。
- 任务物品。
- 一次性消耗品。
- 容器类物品。
- 活体、随从或召唤物相关物品。
- 有特殊 `fueled` 逻辑的物品。
- 通过 prefab 明确写入 Kei 黑名单的物品。

实现时建议采用“黑名单 + 安全组件判断”：

```text
只处理明确拥有普通 finiteuses / armor / fueled 等可维护组件的物品
再排除黑名单 prefab
```

## 生活数据能力定位

第一版生活数据能力分为三类：

| 能力 | 主题 | 作用 |
| --- | --- | --- |
| 初级园艺知识的应用 | 植物学 | 推动作物 / 植物成长阶段。 |
| 初级交易技巧 | 交易 | 制作后概率获得下一次制作消耗减半。 |
| 初级缝补技巧 | 维护 | 一天内逐渐恢复物品栏物品耐久 / 燃料，可突破上限。 |

这三项能力分别改善：

- 资源生产。
- 制作成本。
- 装备维护。

它们共同构成 Kei 生活数据路线的第一批生活质量能力。

## 后续扩展方向

后续想到新的可学习知识时，可以继续按如下格式扩展：

```text
NPC / 世界对象 -> 教材 -> 初级协议 -> 高级协议
```

可扩展方向示例：

| 来源 | 可能学习主题 |
| --- | --- |
| 猪王 | 价值评估、黄金经济。 |
| 薇诺娜 | 工程供能、发电机维护。 |
| 沃姆伍德 | 植物共生、作物生态。 |
| 伏特羊 / 闪电羊 | 电力生态、充电协议。 |
| 远古伪科学站 | 远古结构解析。 |
| 月岛科技 | 月亮能源维护。 |

## 设计边界

- 生活数据不直接提供战斗增伤。
- 生活数据主要提升长期生存效率和生活质量。
- 教材来自 NPC 交互，应尽量保留 NPC 原有交互风格。
- 首次解锁能力时只消耗教材，不触发效果。
- 解锁后的能力应有运行成本，例如数据稳定性、电量、持续时间或概率触发。
- 生活数据能力可以很强，但应避免无条件永久后台生效导致性能或兼容问题。

## 后续待细化

1. 确定三本教材和三项能力的代码名。
2. 确定《初级园艺知识的应用》的影响半径和可影响植物列表。
3. 确定《初级交易技巧》的触发概率和返还材料规则。
4. 确定《初级缝补技巧》的持续时间、恢复频率、恢复量和 500% 上限实现方式。
5. 确定耐久 / 燃料恢复黑名单。
6. 确定教材是物品、蓝图、书籍，还是 Kei 数据终端中的解锁条目。
