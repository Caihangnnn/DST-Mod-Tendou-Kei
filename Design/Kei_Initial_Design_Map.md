# 天童柯伊总体设计

## 角色定位

天童柯伊是学习型机械 AI。角色强度不固定在一套专属装备上，而是通过记录 Boss、解析装备和挂载协议逐步扩展。

## 三维替换

| 原版三维 | Kei 表现 | 初始上限 |
| --- | --- | --- |
| 饥饿 | 电量 | 120 |
| 理智 | 数据稳定性 | 120 |
| 生命 | 机体完整度 | 120 |

每完成一个 Mk 协议槽扩展阶段，三项上限各增加 20。

### 电量

- 沿用饥饿自然下降逻辑；普通食物仅按 20% 效率转化为电量。
- 初始携带 5 个便携电池；每个电池恢复 60 电量。
- 电量为 0 时，默认移动减速并移动受损；天体英雄协议可免除这两项惩罚。

### 数据稳定性

- 常态不自然变化，不受外界理智光环影响。
- 用于主动能力和部分协议消耗；睡眠、休眠等机制可恢复。
- 为 0 时，战斗协议默认失效；织影者协议可覆盖此规则。

### 机体完整度

- 高于 5/6 时每 3 秒自动恢复 1 点。
- 低于 1/6 时每 3 秒降低 1 点；受损状态移动速度降为 1/2，并提示紧急修复。
- 修理工具恢复 50 点完整度。

## 休眠

右键自身进入休眠备份体状态：无法移动、操作物品或施放技能，且不可被选为攻击目标并处于无敌状态。再次右键自身唤醒。

- 每秒最多消耗 1 电量，恢复 3 数据稳定性与 3 完整度；两项均满时不耗电。
- 休眠时战斗协议和解析协议失效，且不触发其固定消耗。
- 电量为 0 后保留 3 秒充电宽限；在薇诺娜发电机范围内休眠时，优先从一台有电发电机获得每秒 3 电量。

## 工程能力

Kei 可制作薇诺娜的投石机、聚光灯、低/高阶发电机、薇机人和遥控器，并拥有便携移动与相关供电适配。由 Kei 放置的发电机与用电器闲置时不耗电，工作时仍正常耗电。
