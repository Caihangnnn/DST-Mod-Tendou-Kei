# 战斗协议清单

战斗协议由数据记录器记录并击败指定目标获得；相同协议重复挂载不叠加。除果蝇王覆盖外，每张每 10 秒消耗 2 数据稳定性。

| 协议 | 记录目标 | 当前效果 |
| --- | --- | --- |
| 独眼巨鹿 | `deerclops` | 攻击加 1 冰冻值；免疫冰冻与过冷。 |
| 独眼晶体巨鹿 | `mutateddeerclops` | 攻击后生成持续 5 秒的紫色时缓圈，非友方单位移速与动画减半。 |
| 附身座狼 | `mutatedwarg` | 按 Z 消耗 10 稳定性喷冷火 5 秒，每 0.5 秒 50 伤害，10 秒冷却。 |
| 熊獾 | `bearger` | 攻击造成衰减群伤。 |
| 装甲熊獾 | `mutatedbearger` | 独立乘区攻击速度 +30%。 |
| 龙蝇 | `dragonfly` | 攻击受控点燃目标；每秒额外造成目标最大生命 0.1% 伤害，持续 30 秒并刷新持续时间；免疫过热与火焰伤害。 |
| 麋鹿鹅 | `moose` | 带电攻击，免疫潮湿。 |
| 克眼 | `eyeofterror`、`twinofterror1`、`twinofterror2` | 右键冲刺至鼠标位置，最大 16 距离，1 秒冷却，路径伤害为普通攻击伤害 2 倍。 |
| 梦魇疯猪 | `daywalker` | 右键选点、左键跳劈；12 距离、3 秒冷却，造成 150 + 目标最大生命 4% 范围伤害、3 秒 10% 移速减速并留坑。 |
| 拾荒疯猪 | `daywalker2` | 免疫僵直、击飞、击退与束缚。 |
| 果蝇王 | `lordfruitfly` | 免除战斗与解析协议的固定资源消耗。 |
| 远古守卫者 | `minotaur` | 攻击有 30% 概率召唤暗影触手、15% 概率暗影囚牢；触手 0.5 秒内置冷却且最多存在 30 秒。 |
| 远古戍卫塔 | `vault_pillar_guard` | 攻速 +20%；近战普通攻击改为 WX-78 式旋转范围攻击。 |
| 战争瓦器人 | `wagboss_robot` | 攻击触发追踪月能轨道打击，20 秒冷却；灼烧为目标最大生命 3% + 50 每秒。 |
| 邪天翁 | `malbatross` | 无视海岸线，可在海面行走。 |
| 克劳斯 | `klaus` | 攻击有 20% 概率抽落即逝灵魂，治疗附近玩家。 |
| 蟾蜍 | `toadstool`、`toadstool_dark` | 攻击有 30% 概率投掷睡袋，0.5 秒内置冷却；免疫催眠。 |
| 蚁狮 | `antlion` | 攻击有 30% 概率在目标脚下生成中心高大沙刺与三个小沙刺。 |
| 蜂后 | `beequeen` | 受攻击时触发威压；默认嘶吼领域使 12 范围生物恐慌 5 秒，3 秒冷却。 |
| 织影者 | `stalker_atrium` | 稳定性为 0 时战斗协议仍生效；攻击有 30% 概率从五个方向影袭，各造成该次伤害 50%。 |
| 天体英雄 | 三阶段 `alterguardian` | 电量为 0 时解析协议仍生效，取消低电量移动惩罚与移动扣血；每 10 秒恢复 10 电量。 |
| 天体后裔 | `alterguardian_phase4_lunarrift` | 三颗宝珠环绕；攻击时加速旋转，每次碰撞造成该次伤害 50% 的额外伤害。 |

战斗协议在死亡、完整度归零和休眠期间全部失效。稳定性为 0 时，除织影者覆盖外全部失效。
