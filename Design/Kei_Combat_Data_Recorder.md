# 战斗数据记录器

## 基本结构

数据记录器由部署包放置，可在任意地点部署，无碰撞体积。未激活时可右键收回为不可堆叠部署包；已激活或完成记录时不可按此方式重置。

记录器使用空白数据记录 CD 与可记录 Boss 建立一次战斗记录。部署后以自身为中心形成竞技场式力场；提交 CD 后开启，Boss 死亡后关闭。

## 记录流程

1. 鼠标持空白 CD 右键指定可记录 Boss，写入绑定目标。
2. 将已绑定 CD 交给记录器；目标必须位于记录器领域内，否则提交失败。
3. 记录器开启力场、光柱和目标扫描特效；Boss 周围出现三个环绕扫描单元。
4. Boss 在领域内死亡后，记录状态变为完成。
5. 左键记录器收取对应战斗协议 CD，记录器回到待机状态。

## 中止与异常

- 记录中右键记录器可中止，关闭力场并返还空白 CD。
- Boss 在领域外死亡不计完成。
- `daywalker`、`daywalker2` 的战败退场状态已按击败处理。
- 空白 CD 未提交记录器前，即使绑定目标死亡也不会报废，可重新绑定。
- 记录器只接受已绑定空白 CD；未绑定时会给 Kei 提示。

## 数据复制

启用“允许拷贝协议数据”后，空白 CD 可右键已完成战斗 CD 复制一份；复制消耗空白 CD。
